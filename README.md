# Very Small CLI using Vendure

This is a very small CLI in Java using the e-commerce framework [Vendure][vendure] as a service. This serves as an example for the software engineering
lecture (_Génie logiciel_) 2026.

In order to use this, you must have [Vendure][vendure] installed and running
(see [Vendure Documentation - Installation][vendure-install]).

[vendure]: https://github.com/vendurehq/vendure
[vendure-install]: https://docs.vendure.io/current/core/getting-started/installation
